Web200 BabyInjection

感谢@一叶飘零师傅 @pcat师傅

这里有一篇关于with rollup的文章，和这题神似。

题目源码：

payload:(POST)

  • with rollup : mysql中把group by的结果进行汇总 具体用法可参看这里
  • select * from table_name group by a with rollup会让查询结果增加一个a为NULL的行
  • select * from users where username='admin' group by passwd with rollup就回产生一个passwd为NULL的行，再用limit它取出来，然后传入一个空的passwd就可以NULL==NULL，绕过验证了。
  • 如果不知道username，则可以使用username=" or 1=1，本题里虽然or被过滤，但还可以使用||。而文章中还给了另一种方式，username="=0。所以给出第二种payloadusername='=0 group by passwd with rollup having passwd <=> null#&passwd=。
  • 若逗号被过滤不能limit n,1，则可用limit 1 offset 1绕过。
  • 而本题，limit被过滤，group by的下一条可用语句是having，则构造having语句来找到passwd=NULL的那一行。
  • having语句生效是因为where语句为true，若直接passwd=NULL，会由于mysqlnull=null返回null而null。故改用<=>，mysql中null<=>null时，返回1，所以有group by passwd with rollup having passwd <=> null
